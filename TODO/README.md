# TODO APP

**Davide Xu**

## Descrizione
Si è sviluppata un’applicazione Todo che permette all’utente di gestire una lista di attività.
L’utente può aggiungere nuovi todo tramite il pulsante “+”, segnandoli come completati tramite una checkbox e cancellarli.
Ogni todo può essere modificata, in modo l’app semplice da usare.

## Scelte di sviluppo
- **Gestione dello stato :** utilizzo di setState per mantenere il codice semplice e compatibile con ambienti online.

- **Funzionalità :** implementazione di aggiunta, modifica, completamento ed eliminazione dei todo.

- **Interfaccia grafica :** design minimale basato sui widget Material di Flutter per garantire chiarezza e usabilità.
